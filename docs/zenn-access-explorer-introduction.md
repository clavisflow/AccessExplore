Access不要の確認アプリ「アクセスエクスプローラー」を作りました。

https://accessexplorer.clavisflow.net/

先日、Excelファイルの差分を確認する「エクセルコンペア」を作りました。

https://zenn.dev/clavisflow/articles/74e5c1b90eab75

今回はそのAccess版に近い位置づけです。

差分を見るアプリではありませんが、「このAccessファイルには何が入っているのか」をブラウザ上でざっと確認できるAccessビューアです。

Macしか手元になかったり、Accessライセンスがなくても中身を確認できます。

もちろんWindows環境でも、Accessを起動せずに素早く構造を確認したい場面で役立ちます。

## 何をするツールか

「このAccess、どんな作りになっているんだろう？」と思ったことはありませんか？

- Macしか手元にない、Accessライセンスがない環境で、`.accdb` や `.mdb` の中身を見たい
- フォーム、レポート、マクロ、モジュールの数だけでも先に知りたい
- テーブルが多くて全体像が分からない
- クエリが大量にあり、どれが何をしているのか追いづらい

アクセスエクスプローラーは、Accessファイルを選択するだけで、テーブルやクエリ、フォーム、レポートなどの構造を一覧で確認できます。

テーブルの実データもプレビュー可能で、CSVによる全件出力も可能です。

また、HTMLレポートはそのままAccess仕様書として利用できます。

## 使い方

使い方はシンプルで、`.accdb` または `.mdb` ファイルをドラッグ＆ドロップするだけです。

最大300MBのAccessファイルに対応しています。

![](https://static.zenn.studio/user-upload/4fd45c4375a0-20260628.png)

解析が終わると、以下のような情報を確認できます。

- Accessバージョン、ファイルサイズ、テーブル数、クエリ数
- テーブルごとのレコード件数と列数
- テーブル定義、列、インデックス、制約
- クエリ一覧とSQL
- フォーム、レポート、マクロ、モジュール
- リレーション
- リンクテーブル

## サーバーにアップロードしない

このツールの大きな特徴は、Accessファイルをサーバーにアップロードしないことです。

業務で使われているAccessファイルには、顧客情報、見積、受注、在庫、社内管理データなどの機密情報が含まれていることがあります。

アクセスエクスプローラーでは解析をブラウザ内で行うため、ファイル内容が外部サーバーへ送信されることはありません。

## まず全体像を見る

Accessファイルを開いた直後は、概要画面で全体像を確認できます。

![](https://static.zenn.studio/user-upload/69d57e23fb81-20260628.png)

たとえば以下のような情報です。

- テーブルはいくつあるか
- レコード件数が多いテーブルはどれか
- クエリ、フォーム、レポートはいくつあるか
- マクロやモジュールが含まれているか

最初にざっと「規模感」をつかむのに役立ちます。

## テーブルとレコード件数を見る

テーブル画面では、テーブル名、列、データ型、レコード件数を確認できます。

![](https://static.zenn.studio/user-upload/d599dde8bd05-20260628.png)

レコード件数順、名前順、列数順で並べ替えられるので、まず大きいテーブルから確認できます。

テーブルごとの実データもプレビューできます。
画面上では先頭50件までを表示し、必要であればテーブル単位で全件CSV出力できます。

## クエリSQLを見る

クエリ画面では、保存されているクエリ名とSQLを確認できます。

![](https://static.zenn.studio/user-upload/03a2c82ea575-20260628.png)

アクセスエクスプローラーでは、クエリを以下のような種類に分けて見られます。

- SELECT
- UPDATE
- DELETE
- INSERT
- MAKE TABLE
- CROSSTAB
- OTHER

更新系クエリや削除系クエリが含まれているかを先に見ておくと、調査時の注意点を把握しやすくなります。

## フォーム・レポート・マクロ・モジュールを見る

オブジェクト画面ではデータベース構造以外の以下のオブジェクトを確認できます。

- フォーム
- レポート
- マクロ
- モジュール
- リレーション
- リンクテーブル

![](https://static.zenn.studio/user-upload/6d528838e39b-20260628.png)

フォームやレポートの中身を完全に解析するものではありませんが、オブジェクト名と件数が分かるだけでも、Accessアプリとしての規模感がかなり見えます。

## スキーマを見る

スキーマ画面では、テーブルのスキーマを確認できます。

![](https://static.zenn.studio/user-upload/89ea45302547-20260628.png)

テーブル構造の詳細確認や、SQL Server などのデータベースへ移行する際のテーブル設計確認にも役立ちます。

## 結果を出力する

解析結果は、JSONまたはHTMLレポートとして出力できます。

HTMLレポートはそのままAccess仕様書としても利用できるので、レビューや調査資料としても使えます。

![](https://static.zenn.studio/user-upload/bcd3a6fe3eca-20260628.png)

## こんなAccessに向いている

- 前任者から引き継いだAccessの中身をざっと確認したい
- 古い `.mdb` ファイルの構造を見たい
- Accessをインストールしていない端末で中身を確認したい
- テーブル数やクエリ数を先に把握したい
- レコード件数が多いテーブルを探したい
- フォーム、レポート、マクロ、モジュールの有無を確認したい
- Access資産の整理前に棚卸ししたい

Accessは業務担当者が長年運用しているケースも多く、年月とともにテーブルやクエリ、フォームが増えて複雑になりがちです。

アクセスエクスプローラーは、そうしたAccessファイルの中身を、最初にざっと見える化するためのアプリです。

## まとめ

アクセスエクスプローラーは、Accessファイルの構造をブラウザ上で確認するAccessビューアです。

Accessをインストールしていない環境でも、テーブル、クエリ、フォーム、レポート、マクロ、モジュール、リレーション、リンクテーブルなどを一覧できます。

ファイルをサーバーにアップロードせずに試せるので、気になるAccessファイルがあれば一度確認してみてください。
